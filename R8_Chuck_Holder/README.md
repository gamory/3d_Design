## R8_Chuck_Holder

Designed around a Grizzly 3-16mm R8 Jacobs style chuck, specifically the one that ships with their G0781 mill.
  
Design details:  
Software used : Fusion 360  
Slicer : Prusa Slicer  
Printer : Prusa i3 Mk3s  
Filament : Prusament Galaxy Silver  
Layer size : 0.1  
Supports : None  
Infill : 15%
